# Contributing to Awesome Web Scraping

Thank you for contributing! This list is community-maintained.

## Guidelines

### Adding a Tool

1. Fork this repository
2. Add your tool to the appropriate category in `README.md`
3. Use the existing table format
4. Submit a pull request

### Requirements

- Tool must be **actively maintained** (updated within last 12 months)
- Tool must be **open source** or have a **free tier**
- Tool must be **relevant** to web scraping
- Description must be **concise** (max 80 characters)

### Format

```markdown
| [Tool Name](https://example.com) | Language | Brief description |
```

### Categories

- **Scraping Frameworks** — Full-featured crawling libraries
- **Browser Automation** — Headless browser control
- **Anti-Detection** — Bot protection bypass
- **Data Extraction** — HTML parsing libraries
- **Proxy Management** — IP rotation tools
- **Scheduling** — Job schedulers

## Code of Conduct

Be respectful. No spam. No self-promotion without disclosure.
